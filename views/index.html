API Project: URL Shortener Microservice

User Story:

  1. Enter a URL below and we will provide you with a shortened URL in the JSON response.
  2. If an invalid URL is passed that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}
  3. When you visit the shortened URL, it will redirect you to your original link.
    To visit the URL using your shortened URL, type this [this_project_url]/api/shorturl/[short_url] and it will redirect you.
